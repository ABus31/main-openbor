最初に

基本性能として、ジャンプの垂直方向の速度が上っています（地上速度は同じ）。
HPが1.2倍とやや増加しています。

ガード不能斬りのモーションが両手の直突きなのが、好きじゃないので、グラフィックが片手平突きになっています
（これは完全な趣味ですヽ(´―｀)ノ）。

とにかく、かっこよく「誠」で決めることに重きを置いたキャラですので、それを堪能してくださいｗ

追加内容

1.虚空殺を昇華対応に

虚空殺を真狼牙で昇華し、ひらう事が可能です。


2.超必→潜在の潜在昇華を可能に

真狼牙→最終狼牙の潜在昇華が可能です。

また、「誠」を味わうために、真狼牙でフィニッシュした時は、
ゲージの本数とコマンド入力がしてあれば、
KO後にゲージ消費なしで最終狼牙をきめて「誠」を堪能可能です(ﾟдﾟ)


3.コマンド投げを追加

63214+X　でコマンド投げが可能。

コマンド投げは虚空殺もしくは真狼牙でキャンセルし、ひらう事が可能です。
また、コマンド投げで向きが入れ替わりコマンドの入力方向が分かりにくくなりやすいので、
コマンド投げをキャンセルして虚空殺を出すときだけ、どちら向きでも技が出るようになっています。


3.5.コマンドの追加

従来のコマンドに加えて、狼牙は236+x、虚空殺は623+a（強はb）、疾空狼牙は2626+a でも出すことが可能です。


4.ガードキャンセルを追加

412+y　で、ゲージ1消費でガードキャンセルが出ます。

出がけ無敵の斬り攻撃ですが、ヒットした時の挙動はオリジナルとは異なり、
地上の相手に当てたときはノックバック、空中の相手に当てた時は空中に打ち上げます。また、必殺技でのキャンセルが可能です。
（このため、かなり強力な技となっています。バランス的にはキャンセル不可の方が良いかもしれませんが、お遊びキャラですので・・・）


5.パワーゲージの自動回復

基本的にゲージがない状態だと、実はほとんどノーマルのキャラと性能は変わりません。
なので、ゲージの自動増加が付いています。また、ゲージを有効利用するため最大数が6本になっています。

結果的に、このゲージの自動増加のためこのキャラクターはかなり強キャラといえます。
（逆にこの特性をなくせば、普通のキャラと大きな変わりがなくなります。）

M_washizuka.cns　の最後付近にある

[state -3, switch]
type = Varset
trigger1 = time = 1
trigger1=roundstate=0
var(57) = 1 ; 0ならパワーゲージの自動回復なし、1なら自動回復ありです。

の、var(57) = 1　を　var(57) = 0
にすれば、ゲージの自動回復をなせますので、お好みで調整ください。


6.いわゆる殺意キャラのような隠し剣質の追加

Pal9（スタート＋Ｃ）とPal12（スタート＋Ｚ）のカラーが隠し剣質となっています。

隠し剣質は、いわゆる力＋技のいいところ取りです。力の攻撃力と昇華、技の連殺陣と乱舞奥義が使用可能で、
極との一番の違いとして防御力低下のデメリットがありません。
ゲージの自動上昇量が通常の1.5倍、ラウンド開始時にゲージ2本分追加があり、瀕死時のＨＰの自動回復もあります。
また、ガードキャンセル技の出がかりに弾き判定が付いています。

反則的な強さを持つ一部のキャラ程ではないですが、この隠し剣質は相当な強さです。（厨っぽいともいえますヽ(;´Д｀)ノ）
お遊びと思って、暴れてもらうのがいいと思います。
むしろ、そういう遊び方をするための改変キャラですので、この剣質こそがこのキャラの本道といえるかもしれません( ﾟДﾟ)

AIについて

AIは11段階のレベル式AIです。
AI.cns　の中の

[State -3, AI]
Type = VarSet
TriggerAll = ((Var(59) = 0) && (RoundState != 3))
trigger1 = command = "AI_01"
　　　　　〜
　　　　　中略
　　　　　〜
trigger50 = command = "AI_50"
Trigger51 = Numhelper(10000)
Trigger51 = helper(10000),var(59)
V = 59
Value =11 ; 1〜11でレベルを決めてください。1が弱く、11が最強です。
IgnoreHitPause = 1

この、Value =　の後に数字1〜11を入れてください。
レベル1が最弱で、11が最強です。

連続技

基本的な流れは、昇華可能剣質なら

・通常技→狼牙→直式→真狼牙→最終狼牙（ゲージがないなら弱俊殺）
・通常技→狼牙→上式→虚空殺→真狼牙→最終狼牙（ゲージがないなら弱俊殺）

の2種類です。昇華できない剣質「技」のみは、通常技から直接に真狼牙をつないでください。
また、ゲージさえあれば乱舞奥義の使える剣質なら、（通常技→乱舞奥義→）を頭にはさむ事が出来ます。


・コマンド投げ→虚空殺→（昇華可能なら)真狼牙→最終狼牙
